```text

```

验收对象：

| 模块 | 范围 |
|------|------|
| 前端 | 5 个页面：首页、打卡页、随心测、错题本、个人中心 |
| 后端 | standalone dev-server 与正式服务接口协议 |
| 数据 | OCR 词库 CSV/JSON、standalone 30 条测试词库、正式导入接口 |
| 文档 | PRD、用户故事、测试用例、开发方案、API 文档、部署方案、差异清单 |

## 六、上线前风险

| 风险 | 影响 | 处理 |
|------|------|------|
| OCR 词库当前为 755 条，不是完整 800 条 | 影响产品名与实际词库一致性 | 上线前由用户确认最终词库，或补齐缺失词条 |
| standalone 使用内存数据 | 重启丢失数据 | 上线必须切 MySQL 正式服务 |
| 微信登录尚未在真实小程序环境验收 | 用户无法正式登录 | 上线前必须跑微信 code 登录 |
| 小程序必须使用 HTTPS 合法域名 | 本地接口无法上线使用 | 部署后配置小程序后台域名 |
