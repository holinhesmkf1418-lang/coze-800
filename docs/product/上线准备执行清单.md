## 五、上线顺序

推荐按以下顺序推进：

1. 用户确认服务器、域名、AppID/Secret、词库条数。
2. 后端部署 MySQL 与正式服务。
3. 后端导入正式词库。
4. 前端切换生产 `baseURL` 和 `loginMode='wechat'`。
5. 微信后台配置 request 合法域名。
6. 微信开发者工具真机预览。
7. PM 执行上线前回归验收。
8. 通过后准备小程序提审。
